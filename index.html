Math item markup 03 Jan 2014

The markup to use when editing math items is based on Markdown. All the markup of Markdown is supported, except for inline HTML, links and images.

On categories 03 Dec 2013

A category, as used on teoremer, can be seen as a tag with another category as parent. A special root tag is used to represent top-level categories and no cycles are allowed. This way all categories represent a rooted tree, or heirarchy, of tags.

Introducing teoremer 03 Nov 2013

If you distill any book or article on some mathematical subject, then what you are left with is usually a lot of notation, definitions, algorithms, theorems, lemmas, corollaries, and proofs.

teoremer is a collection of such mathematical items.
